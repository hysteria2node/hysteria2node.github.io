点击某个节点，Snap会开始尝试连接，连接成功后状态显示为“已连接”。此时，你的所有网络流量都将通过该代理节点转发。

Snap还支持“分应用代理”，即你可以选择只让特定App（如浏览器）走代理，其它App走直连网络。

开启方式：

设置 > 应用管理 > 选择需要代理的App > 勾选启用

安卓Snap使用中的常见问题解答

Q1：Snap连接不上，怎么办？

  • 清除Snap缓存并重启App。

  • 检查是否被系统电池优化限制后台运行。

Q2：Snap安全吗？

答：只要你使用的是可信节点（如自建或正规机场提供），Snap本身不会泄露隐私。但要避免使用来路不明的配置链接，防止遭遇钓鱼节点或数据劫持。

Q3：Snap支持哪些协议？

答：主要支持：

因此，无论你的节点提供什么类型的服务，大多数情况下都能通过Snap连接。

Q4：Snap对安卓系统版本有要求吗？

答：Snap基本支持Android 5.0以上版本，部分旧机型可能运行缓慢或启动失败，建议使用Android 8.0及以上版本获取最佳体验。

Q5：如何更新Snap？

FAQ

Sing-box 是什么？
Sing-box 是新一代多协议代理框架，支持 VMess、VLess、Trojan、Hysteria、ShadowTLS 等协议。它比传统 V2Ray 更轻量化、模块化，并针对移动端和 Linux 系统进行了性能优化，受到开发者和高级用户的欢迎。
QuantumultX 如何实现按应用分流？
QuantumultX 可以根据不同应用程序的流量类型设置分流规则，通过策略组选择最优节点。这样可保证游戏、视频和浏览器流量经过最优节点，提高访问速度和网络稳定性。
Sing-box 支持哪些类型的分流规则？
Sing-box 支持域名、IP、端口、协议类型、用户标签等多种分流规则。结合策略组，可实现精细化流量管理，满足视频加速、广告屏蔽和游戏优化等多样化需求。
Mellow 是否支持 IPv6 流量代理？
Mellow 支持 IPv6，只需在配置文件中启用 IPv6 DNS 解析并确认操作系统开启 IPv6 路由功能。部分节点若不支持 IPv6，则相关请求会自动回退到 IPv4，保证兼容性。
SSR 如何选择合适混淆提高稳定性？
选择混淆方式时应根据网络环境和封锁程度，例如 tls1.2_ticket_auth 可模拟 HTTPS 流量。合理组合协议和混淆可提升节点稳定性和抗封锁能力，确保长期科学上网体验。
Vmess 节点可以通过 CDN 吗？
可以。Vmess 可通过 WebSocket 与 CDN 结合使用，只需将传输方式设置为 WS 并配置 host 与 path，即可实现流量中转。这样既能隐藏真实服务器，又能提升连接速度。
Trojan 的伪装域名为何重要？
伪装域名必须是真实可访问的网站，且证书有效。它用于伪装流量，使 Trojan 的数据在外观上完全像正常 HTTPS 请求，从而有效降低被封锁和 DPI 检测的风险。
Mihomo 是什么？
Mihomo 是 Clash Premium 的开源替代项目，兼容原版 Clash 配置文件并扩展了部分新功能。它性能优异、内存占用低，适用于希望自定义构建代理环境的高级用户。
Sing-box 如何实现流量精细分流？
通过定义 routing 规则和策略组，Sing-box 可以根据域名、IP、端口或协议类型精细分流流量。例如视频走高速节点，网页走备用节点，实现流量优化和网络加速。
SSR 的端口复用有什么风险？
SSR 支持多端口配置以复用服务，但若端口过多或设置冲突，可能被扫描工具识别或误判为异常服务。建议保持配置简洁，并合理分配端口范围以减少暴露风险。
